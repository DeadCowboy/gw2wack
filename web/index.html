Introduction

Welcome to WACK, the World Auxilary Command Konsole! This web application uses the official Guild Wars 2 API to bring you the latest status of all WvW (World vs World) match ups in the game. Using responsive web design techniques, this application will conform to fit on any HTML5 compatible browser. No need for a native app for your computer desktop or mobile phone, just use your web browser!

Instructions

From the world select menu, simply click on your world server. The app will then redirect you to the correct URL for your current match up. You can bookmark this page for quick access in the future. Every 30 seconds the app will update the match details.

Objective Icons

Each objective icon represents a type of fortification in WvW, as follows:

  • Stonemist Castle - The biggest structure in WvW strategically positioned in the center of the Eternal Battlegrounds.
  • Keeps - Three keeps are found in each WvW map. These massive fortifications employ inner and outer gates that must be breached to be taken.
  • Towers - Smaller fortifications defended by a single gate and wall.
  • Camps - Generating supply and delivering it to nearby towers and keeps, each battleground has six camps. They are defended only by a small group of guards and include no walls or gates.

Directional Icons

The directional arrows on the right of each objective indicate the location of the fortification on the map, as follows:

  • East side of the map.
